Manual do Conector Infor

O conector Infor tem o objetivo de extrair dados do DataLake ou DataFabric, transformá-los e enviá-los ao portal da Compliance Soluções.
Você pode acessar a Documentação Técnica e ver o mapeamento com as regras de transformação de dados após a extração.

Após fazer login, o acesso será redirecionado para tela de dashboard, conforme demonstra a imagem a seguir.

Menu Lateral

Como podemos ver, o menu lateral possui poucos botões.

O primeiro botão com ícone de casa é para acessar o dashboard, o segundo botão que chamamos de menu hambúrguer (3 traços) é para acessar o menu principal e o terceiro botão é para trocar o idioma do conector. Dentro do menu hambúrguer há diversas opções que serão exploradas mais adiante.

Os outros 2 botões no menu lateral são para alterar modo claro ou escuro e acessar dados do usuário respectivamente.

Menu Principal

Ao acessar o menu principal veremos 4 opções:

  • Administrativo Geral: serve para configurar o conector. Esse menu nem sempre estará disponível para todos os usuários.
  • Agendamentos: é onde criamos as requisições para o conector fazer as extrações no Infor.
  • Monitoramento NF-e e NFS-e: são opções para acompanhar o faturamento de notas de mercadoria e serviço respectivamente. Essas não aparecem para todos os clientes porque podem existir outros métodos de integração de faturamento que não passam por esse local.

Menu do Usuário

A seguir o menu que é carregado ao clicar sobre a foto do usuário.

Na opção Ajuda, seremos direcionados para uma página com a documentação do conector. Em Meus Dados, é possível ajustar o cadastro do usuário e escolher algumas opções de segurança, teremos mais detalhe na sequência. A opção Segurança serve para realizar a troca da senha.

Meus Dados

A seguir, veremos detalhes da opção Meus Dados. A opção de trocar de Organização e Empresa só aparecerá para o Suporte Compliance. A Autenticação 2FA permite habilitar uma segunda camada de segurança com token de validação baseado em hora, tais como Google ou Microsoft Authenticator.

Agendamentos

Ao acessar a página de agendamento, é possível acompanhar cargas que estão sendo realizadas ou histórico do que já foi executado. Também é simples criar um novo agendamento.

As opções do agendamento são:

  • Frequência: Define-se o agendamento irá ocorrer apenas uma vez ou de forma recorrente. Caso seja recorrente, será aberta opção para escolher a frequência e momento da execução.
  • Objeto de Integração e Tipo de Objeto: Serve para escolher a qual tipo de dado deverá ser carregado, por exemplo, dados cadastrais ou modelos de documentos fiscais.
  • Empresa: indica em qual empresa buscar os dados.
  • Período da Carga: quando a carga tem um período, tais como documento fiscal, inventário ou contábil, é possível determinar o intervalo de datas que deve ser considerado para filtrar e integrar os dados.
  • Número de documento: útil para quando deseja integrar apenas um documento fiscal específico.
  • Caixa de marcação Integrar todos os registros: só está disponível na versão 1 do extrator. Ela é utilizada para reintegrar todos os dados, desde o ID de processo zero quando estiver marcada. Caso esteja desmarcada, apenas os dados com ID de processo nunca recebidos é que serão integrados.
Monitoramento de Notas

Telas de monitoramento de notas é possível ver as notas que estão sendo recebidas do ERP para serem enviadas ao Compliance para realizar a autorização junto à Sefaz (estadual ou municipal). Também é possível acompanhar o retorno dos dados para o ERP, inclusive forçando nova tentativa de comunicação se for necessário.
